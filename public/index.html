图像处理 python 直方图均衡化

直方图均衡化

  • 1.实验原理

    用直方图变换方法进行图像增强，通过改变图像的直方图来概念图像中像素的灰度，以达到图像增强的目的。

  • 2.实验步骤

    1、对图像进行灰度统计，求灰度统计直方图。

    2、对灰度统计直方图进行归一化。

    3、求累积分布函数，求累积分布直方图。

    4、对累积直方图各项进行取整扩展tk=int[(L-1)tk + 0.5].

    5、确定映射对应关系，根据映射关系计算均衡化直方图。
